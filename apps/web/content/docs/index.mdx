## Overview

Devo is a Rust-based coding agent. This web project uses Next.js, Tailwind CSS 4, and Fumadocs for the documentation surface.

## Local Development

Run the web app from `apps/web`:

```bash
nvm use 22
npm run dev
```

The documentation route is available at `/docs`.
