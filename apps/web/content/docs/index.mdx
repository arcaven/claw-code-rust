## Overview

Devo is an open coding agent for developers who want a terminal-first workflow,
provider flexibility, and an inspectable alternative to closed agent tools.

The model system is built from three separate concepts:

- Supported model definitions: built-in capability metadata shipped with Devo.
- Provider entries: reusable user-defined connection records.
- Model bindings: invocable links between a supported model and a provider.
